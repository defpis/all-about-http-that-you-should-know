# HTTP 数据协商

## 相关请求头

以下为客户端发送设置

| 请求头          | 作用                     |
| --------------- | ------------------------ |
| Accept          | 指定接受数据的类型       |
| Accept-Encoding | 指定接受数据的压缩方式   |
| Accept-Language | 指定接受数据的语言       |
| User-Agent      | 指定接受数据的客户端类型 |

以下为服务端返回设置

| 请求头           | 作用                   |
| ---------------- | ---------------------- |
| Content-Type     | 指定返回数据的类型     |
| Content-Encoding | 指定返回数据的压缩方式 |
| Content-Language | 指定返回数据的语言     |
